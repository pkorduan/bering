Sicherungen

Sicherungen anlegen

Wenn Sie vor jeder Änderung an der Datenbank automatisch eine Sicherungskopie anlegen möchten, wählen Sie unter Einstellungen ein Verzeichnis, z.B. C:\Benutzer und ein Dateinamen z.B. sicherung_bering aus uns stellen Sie die Anzahl der Sicherungen ein.
Ist die Anzahl höher als 0, wird die aktuelle Datenbank in dem gegegebenen Verzeichnis mit einem Zeitstempel kopiert, z.B. sicherung_bering_2018-05-12_12:34:21.db Übersteigt die Anzahl der Kopien die vorgegebene Anzahl, werden die jeweils die ältesten gelöscht.
Ist z.B. die Anzahl auf 10 festgelegt, kann man genau 10 Änderungen rückgängig machen.

Sicherungen wiederherstellen

Zur Wiederherstellung einer gesicherten Datenbank ist die Anwendung zu schließen und die Datei bering.db durch die Sicherungsdatei z.B. sicherung_bering_2018-05-12_12:34:21.db zu ersetzen. Anschließend kann die Anwendung wieder gestartet werden und der Stand der Sicherung ist aktuell.

Export

Zum Exportieren von Daten muss vorher unter Einstellungen ein Verzeichnis angegben werden.
Der Export erfolgt unter dem gleichnamigen Menüpunkt durch klick auf die Fundart. In Runden Klammern ist angegeben wie viele neue Datensätze vorhanden sind, bzw. exportiert werden können.
Nach dem erfolgreichen Export wird der vergebene Dateiname angezeigt. Die Dateinamen setzen sich wie folgt zusammen:
Kürzel für Fundart (B Beringung, W Wiederfunde, F Fremdfunde), Beringernummer, Datum, Zeit, .sdf

Datenbank-Verwaltung

Das Anlegen neuer Nutzer erfolgt direkt in der Datenbank. Zur Verwaltung der Daten in der Datenbank wird das Programm SQLLiteStudio verwendet. Es kann hier heruntergeladen werden. Nach der Installation und dem Start des Programmes muss man eine neue Datenbankverbindung zur bering-Datenbank herstellen. Dazu wählt man den Menüpunkt Datenbank / Datenbank hinzufügen

Anschließend kann man die Datei mit der Datenbank auswählen. Mit einem Klick auf den Ordner-Icon unter Datei gelangt man in Datei-Dialog. Die Datei bering.db befindet sich in dem Verzeichnis in dem das bering Programm entpackt wurde und in dem Verzeichnis unter resources/app/app/db. Ist die Datei ausgewählt kann man die Verbindung testen und dann OK klicken.

Die Datenbank bering sollte jetzt im Linken Fenster von SQLLiteStudio zu sehen sein und kann durch doppelklick aufgeklappt werden. Klappen Sie auch Tabellen auf. Ein Doppelklick auf Nutzer öffnet die Entwurfsansicht der Tabelle. Um die Daten zu sehen und bearbeiten zu können im rechten Fenster den Reiter Daten auswählen.

Über den grünen Plus-Button kann nun ein neuer Nutzer hinzugefügt werden. Als Passwort sollte per Paste + Copy zunächst das gleiche wie von Admin verwendet werden. Es kann später durch den Nutzer geändert werden.
